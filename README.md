# Сегментация автомобиля на части

## Описание проекта
Проект по семантической сегментации изображений автомобилей с использованием архитектуры U-Net и предобученного энкодера ResNet18. Модель выделяет 4 класса объектов: фон, кузов, диски и стекла.

## Цель проекта
Построить модель глубокого обучения, которая на основе входного изображения предсказывает пиксельную маску с разделением на классы.

**Целевые метрики:**
- IoU
- Dice Loss
- F1-score

## Структура и этапы работы

### 1. Предобработка данных
- Создан кастомный класс `SegmentationElCar` для загрузки данных
- Реализовано преобразование RGB-масок в one-hot кодировку
- Цветовая карта классов загружается из внешнего файла

### 2. Аугментации данных
Применены аугментации для улучшения обобщающей способности модели:
- Resize (256×256)
- HorizontalFlip
- ShiftScaleRotate (сдвиг, масштабирование, поворот)
- CoarseDropout
- RandomBrightnessContrast

### 3. Архитектура модели
- **Архитектура:** U-Net
- **Энкодер:** ResNet18 (предобучен на ImageNet)
- **Функция активации:** Softmax2d
- **Количество классов:** 4

### 4. Обучение модели
**Конфигурация:**
- Оптимизатор: Adam (lr=0.0001)
- Функция потерь: Dice Loss
- Метрики: F1-score, IoU
- Эпохи: 60
- Batch size: 32

**Стратегия:**
- Снижение learning rate каждые 15 эпох

### 5. Визуализация результатов
Разработаны функции для визуализации аугментаций, графиков обучения и предсказаний модели.

## Результаты
